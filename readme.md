# HTML/CSS/JS hosting on GitHub (pages)

Check out the [demo](https://codepo8.github.io/code-hosting-demo/example/) and [see the source](https://codepo8.github.io/code-hosting-demo/example/index.md). It is that easy! 

1. Put your files in a folder
1. Add an `index.md` and point the includes to the files. 
1. Profit.
